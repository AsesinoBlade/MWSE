# Audio Mix Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.audioMixType.*` namespace. For example, `tes3.audioMixType.Voice` has a value of `1`.

ID  | Audio Mix Type
--- | ------------------
0   | Master
1   | Voice
2   | Effects
3   | Footsteps
4   | Music

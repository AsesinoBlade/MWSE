# Sound Mix Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.soundGenType.*` namespace. For example, `tes3.soundGenType.roar` has a value of `1`.

ID | Sound Mix Type
-- | ---------------------
0  | Master
1  | Voice
2  | Effects
3  | Footsteps
4  | Music

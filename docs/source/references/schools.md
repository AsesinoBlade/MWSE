# Schools

!!! tip
	These values are available in Lua by their identifier in the `tes3.school.*` namespace. For example, `tes3.school.minor` has a value of `1`.

ID | School
-- | ----------------
0  | Alteration
1  | Conjuration
2  | Destruction
3  | Illusion
4  | Mysticism
5  | Restoration

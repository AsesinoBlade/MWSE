
xBitNot
========================================================

**Parameters:**

- ``long`` **leftValue**: Value for the bitwise operation.

**Returned:**

- ``long`` **result**: 1 if the bitwise operation returns true.

This function performs a NOT bitwise operation, and returns its result.

==== =========
A    xBitNot A
==== =========
0000 1111
0101 1010
1100 0011
1111 0000
==== =========

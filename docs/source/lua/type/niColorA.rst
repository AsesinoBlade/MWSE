niColorA
====================================================================================================

An object that represents a RGB color with alpha channel support.

`g <niColorA/g.html>`_ (`number`_)

`green <niColorA/green.html>`_ (`number`_)

`r <niColorA/r.html>`_ (`number`_)

    Alias for the red value of the color.

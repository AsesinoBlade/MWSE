tes3game
====================================================================================================

A core game object used for storing game settings.

Properties
----------------------------------------------------------------------------------------------------

`parentWindowHandle <tes3game/parentWindowHandle.html>`_ (`number`_)
    The Windows HWND for the parent window.

`playerTarget <tes3game/playerTarget.html>`_ (`tes3reference`_)
    The current activation target.

`renderDistance <tes3game/renderDistance.html>`_ (`number`_)
    The game's rendering distance.

`screenShotsEnabled <tes3game/screenShotsEnabled.html>`_ (`boolean`_)
    State on if screenshots are enabled.

`screenX <tes3game/screenX.html>`_ (`number`_)
    The screen's horizontal position.

`screenY <tes3game/screenY.html>`_ (`number`_)
    The screen's vertical position.

`soundQuality <tes3game/soundQuality.html>`_ (`number`_)
    Sound quality level.

`voice <tes3game/voice.html>`_ (`number`_)
    Mix volumes for voices.

`volumeEffect <tes3game/volumeEffect.html>`_ (`number`_)
    Mix volumes for effects.

`volumeFootsteps <tes3game/volumeFootsteps.html>`_ (`number`_)
    Mix volumes for footsteps.

`volumeMaster <tes3game/volumeMaster.html>`_ (`number`_)
    Mix volumes for all sounds.

`volumeMedia <tes3game/volumeMedia.html>`_ (`number`_)
    Mix volumes for music.

`windowHandle <tes3game/windowHandle.html>`_ (`number`_)
    The Windows HWND for the window.

`wireframeProperty <tes3game/wireframeProperty.html>`_ (`niProperty`_)
    The reused wireframe property, appled when toggling wireframe rendering.

`worldSceneGraphRoot <tes3game/worldSceneGraphRoot.html>`_ (`niNode`_)
    Access to the root of the scene graph.

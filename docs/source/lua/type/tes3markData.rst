tes3markData
====================================================================================================

A simple structure to hold mark/recall data.

Properties
----------------------------------------------------------------------------------------------------

`cell <tes3markData/cell.html>`_ (`tes3cell`_)
    The cell to recall to.

`position <tes3markData/position.html>`_ (`tes3vector3`_)
    The position to recall to.

`rotation <tes3markData/rotation.html>`_ (`number`_)
    The rotation to use when recalling.

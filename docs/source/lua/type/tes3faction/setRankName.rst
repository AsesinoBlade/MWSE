setRankName
====================================================================================================

This function changes the player-facing name of a given rank in the faction.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

rank (`number`_)
    A rank number in range [0, 9] inclusive.

name (`string`_)
    A new name to set.

.. _`string`: ../../../lua/type/string.html

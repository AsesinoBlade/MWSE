getRankName
====================================================================================================

This function fetches the player-facing name of a rank in the faction.

Returns
----------------------------------------------------------------------------------------------------

`string`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

rank (`number`_)
    A rank number in range [0, 9] inclusive.

.. _`string`: ../../../lua/type/string.html

isMouseButtonPressedThisFrame
====================================================================================================

Returns true if a mouse button was up last frame, and down this frame.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

button (`number`_)
    The mouse button to test. Must be between 0 and 7.

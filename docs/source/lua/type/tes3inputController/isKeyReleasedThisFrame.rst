isKeyReleasedThisFrame
====================================================================================================

Checks to see if a given scan code is released, and was pressed last frame.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

key (`number`_)
    The scan code to test. Constants available through ``tes3.scanCode``.

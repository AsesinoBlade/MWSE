tes3packedColor
====================================================================================================

A densely packed structure for single-byte red, green, blue and alpha values.

Properties
----------------------------------------------------------------------------------------------------

`a <tes3packedColor/a.html>`_ (`number`_)
    The alpha color value.

`b <tes3packedColor/b.html>`_ (`number`_)
    The blue color value.

`g <tes3packedColor/g.html>`_ (`number`_)
    The green color value.

`r <tes3packedColor/r.html>`_ (`number`_)
    The red color value.

createDivider
====================================================================================================

Creates a textured horizontal divider line, as used in the magic and stat menus. Scales automatically to the container width.

Returns
----------------------------------------------------------------------------------------------------

`tes3uiElement`_.

Parameters
----------------------------------------------------------------------------------------------------

id (`number`_)
    Optional. A registered identifier to help find this element later.

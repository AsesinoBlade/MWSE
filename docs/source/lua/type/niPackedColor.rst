niPackedColor
====================================================================================================

An object that represents a RGB color with alpha channel support. Unlike other color values, this is compressed into 4 bytes. Values are not from 0 to 1, but from 0 to 255.

Properties
----------------------------------------------------------------------------------------------------

`a <niPackedColor/a.html>`_ (`number`_)
    The alpha value of the color.

`alpha <niPackedColor/alpha.html>`_ (`number`_)
    The alpha value of the color.

`b <niPackedColor/b.html>`_ (`number`_)
    The blue value of the color.

`blue <niPackedColor/blue.html>`_ (`number`_)
    Alias for the blue value of the color.

`g <niPackedColor/g.html>`_ (`number`_)
    The green value of the color.

`green <niPackedColor/green.html>`_ (`number`_)
    Alias for the green value of the color.

`r <niPackedColor/r.html>`_ (`number`_)
    The red value of the color.

`red <niPackedColor/red.html>`_ (`number`_)
    Alias for the red value of the color.

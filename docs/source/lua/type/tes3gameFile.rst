tes3gameFile
====================================================================================================

Represents a loaded ESM, ESP, or ESS file.

Properties
----------------------------------------------------------------------------------------------------

`author <tes3gameFile/author.html>`_ (`string`_)
    The file's author.

`cellName <tes3gameFile/cellName.html>`_ (`string`_)
    The current cell, from a save game.

`currentHealth <tes3gameFile/currentHealth.html>`_ (`number`_)
    The player's health, from a save game.

`day <tes3gameFile/day.html>`_ (`number`_)
    The current day, from a save game.

`daysPassed <tes3gameFile/daysPassed.html>`_ (`number`_)
    The number of days passed, from a save game.

`description <tes3gameFile/description.html>`_ (`string`_)
    The description of the file.

`filename <tes3gameFile/filename.html>`_ (`string`_)
    The simple filename.

`fileSize <tes3gameFile/fileSize.html>`_ (`number`_)
    The size of the file.

`gameHour <tes3gameFile/gameHour.html>`_ (`number`_)
    The current game hour, from a save game.

`highestFormID <tes3gameFile/highestFormID.html>`_ (`number`_)
    No description available.

`masters <tes3gameFile/masters.html>`_ (`table`_)
    An array-style table of the tes3gameFiles that this is dependent upon.

`maxHealth <tes3gameFile/maxHealth.html>`_ (`number`_)
    The player's maximum health, from a save game.

`modifiedTime <tes3gameFile/modifiedTime.html>`_ (`string`_)
    The timestamp that the file was modified.

`month <tes3gameFile/month.html>`_ (`number`_)
    The current month, from a save game.

`path <tes3gameFile/path.html>`_ (`string`_)
    The path to the file.

`playerName <tes3gameFile/playerName.html>`_ (`string`_)
    The player's name, from a save game.

`year <tes3gameFile/year.html>`_ (`number`_)
    The current year, from a save game.

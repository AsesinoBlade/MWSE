potionBrewed
====================================================================================================

The potionBrewed event is raised when the player brews a new potion.

Event Data
----------------------------------------------------------------------------------------------------

object
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

tes3alchemy. Read-only. The potion created by the player.

determinedAction
====================================================================================================

This event fires when an actor has determined an action in a combat session.

Event Data
----------------------------------------------------------------------------------------------------

session
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

tes3combatSession. Read-only. The combat session the action has been determined for.

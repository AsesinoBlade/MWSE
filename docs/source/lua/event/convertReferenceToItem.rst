convertReferenceToItem
====================================================================================================

This event is raised when an in-world tes3reference for an item is about to be converted to fit into a tes3itemStack.

Event Data
----------------------------------------------------------------------------------------------------

reference
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

tes3reference. Read-only. The reference about to be converted.

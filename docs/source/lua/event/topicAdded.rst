topicAdded
====================================================================================================

This event is raised when the player gains access to a new dialogue topic.

Event Data
----------------------------------------------------------------------------------------------------

topic
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

tes3dialogue. Read-only. The topic that was newly added to the player's list of topics.

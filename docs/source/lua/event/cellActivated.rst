cellActivated
====================================================================================================

This event is triggered when a cell is activated.

Related events: `cellActivated`_, `cellDeactivated`_, `cellChanged`_

Event Data
----------------------------------------------------------------------------------------------------

cell
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`tes3cell`_. Read-only. The cell which was activated.

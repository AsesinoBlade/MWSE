buttonPressed
====================================================================================================

The buttonPressed event is unique in that it is invoked only when using tes3.messageBox to present buttons to the player.

Event Data
----------------------------------------------------------------------------------------------------

button
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`number`_. Read-only. The button index, 0-based, that was pressed.


tes3ui.menuMode
====================================================================================================

Checks if the game is in menu mode.

.. code-block:: lua

    if (tes3ui.menuMode()) then
        doThing()
    end

**Parameters:**

- *None*

**Returns:**

- `boolean`_ **result**
    ``true`` if in menu mode.


.. _`boolean`: ../../type/lua/boolean.html

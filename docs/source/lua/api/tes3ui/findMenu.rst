
tes3ui.findMenu
====================================================================================================

Locates a top-level menu through its id.

.. code-block:: lua

    tes3ui.findMenu(id_mymenu)

**Parameters:**

- `UI_ID`_ **menuID**
    The ID of the menu to locate.

**Returns:**

- `Element`_ **result**
    The menu element, or ``nil`` if it was not found.

tes3ui.enterMenuMode
====================================================================================================

Requests menu mode be activated on a menu with a given id.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_. true if the menu was switched, or false if it was already active.

Parameters
----------------------------------------------------------------------------------------------------

id (`number`_)
    No description available.

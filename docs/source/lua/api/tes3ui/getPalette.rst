tes3ui.getPalette
====================================================================================================

Gets a standard palette color. Returns an array containing the RGB color values, in the range [0.0, 1.0].

Returns
----------------------------------------------------------------------------------------------------

result (`table`_)
    No description available.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

name (`string`_)
    The name of the palette color.

json.savefile
====================================================================================================

Saves a serializable table to Data Files\MWSE\{fileName}.json, using json.encode.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters in the following order:

fileName (`string`_)
    No description available.

object (`table`_)
    No description available.

config (`table`_)
    Optional. No description available.

mge.disableHUD
====================================================================================================

Disables a HUD element. If a HUD ID is provided, this is a warpper for MGEDisableHUD, otherwise it is a wrapper for MGENIDDisableHUD.

Parameters
----------------------------------------------------------------------------------------------------

hud (`string`_)
    Optional. No description available.

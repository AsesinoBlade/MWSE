mge.setHUDTexture
====================================================================================================

Sets the texture for a given HUD element. If a HUD ID is provided, this is a warpper for MGEChangeHUDTexture, otherwise it is a wrapper for MGENIDChangeHUDTexture.

Parameters
----------------------------------------------------------------------------------------------------

hud (`string`_)
    No description available.

texture (`string`_)
    No description available.

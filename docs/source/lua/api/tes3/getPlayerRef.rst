
tes3.getPlayerRef
========================================================

**Parameters:**

- *None*

**Returned:**

- ``reference`` **reference**: A `reference`_ to the player.

Returns a reference to the player.

Examples
--------------------------------------------------------

.. code-block:: lua

  -- Show the player's name.
  local player = tes3.getPlayerRef()
  print(player.object.name)

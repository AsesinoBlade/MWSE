tes3.fadeOut
====================================================================================================

Similar to the vanilla FadeOut mwscript command.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters through a table with the given keys:

fader (`tes3fader`_)
    Optional. Defaults to the transition fader.

duration (`number`_)
    Default: ``1.0``. Time, in seconds, for the fade.

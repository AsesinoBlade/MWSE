tes3.loadMesh
====================================================================================================

Loads a mesh file and provides a scene graph object.

Returns
----------------------------------------------------------------------------------------------------

`niNode`_.

Parameters
----------------------------------------------------------------------------------------------------

path (`string`_)
    Path, relative to Data Files/Meshes.

useCache (`boolean`_)
    Default: ``true``. If false, a new object will be created even if it had been previously loaded.

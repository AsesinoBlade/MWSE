tes3.getCell
====================================================================================================

Finds a cell, either by an id or an X/Y grid position.

Returns
----------------------------------------------------------------------------------------------------

`tes3cell`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters through a table with the given keys:

id (`string`_)
    The cell's ID. If not provided, x and y must be.

x (`number`_)
    No description available.

y (`number`_)
    No description available.

tes3.findGMST
====================================================================================================

Fetches the core game object that represents a game setting. While this function accepts a name, it is recommended to use the tes3.GMST constants.

Returns
----------------------------------------------------------------------------------------------------

`tes3gameSetting`_.

----------------------------------------------------------------------------------------------------

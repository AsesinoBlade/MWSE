tes3.addArmorSlot
====================================================================================================

Registers a new slot to be used.

Returns
----------------------------------------------------------------------------------------------------

`boolean`_.

Parameters
----------------------------------------------------------------------------------------------------

Accepts parameters through a table with the given keys:

slot (`number`_)
    A number greater than 10 to configure a slot for.

name (`string`_)
    No description available.

weight (`number`_)
    A stand-in for the armor base weight value, typically controlled by a GMST (e.g. iHelmWeight).
